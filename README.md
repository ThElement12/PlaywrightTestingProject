# Playwright Testing Project

This project contains automated tests built using **Playwright**.

---

## Requirements

Before installing and running the project, make sure you have the following installed:

* **Node.js** (version 18 or higher recommended)
* **npm** (comes with Node.js)

---

## Installation

1. Install dependencies:

```bash
npm install
```

2. Install Playwright browsers:

```bash
npx playwright install
```

3. Add `.env` file in the root folder:

---

## Running the tests

Run all tests:

```bash
npx playwright test
```
---

## Test Reports

After running the tests, you can open the HTML report with:

```bash
npx playwright show-report
```
